# Dowin 성능 Baseline (1차)

## 1. 측정 목적

- 최적화의 효과를 체감이 아닌 수치로 비교하기 위한 기준선 문서
- 이후 변경은 동일 조건에서 재측정하여 p50/p95 개선 여부를 기록

## 2. 측정 범위

- 우선 엔드포인트
  - `GET /api/dashboard/team`
  - `GET /api/scoreboards/:id/logs/weekly`
  - `GET /api/scoreboards/:id/logs/monthly`
- 환경
  - Cloudflare Workers + D1 (운영과 동일한 런타임 기준)
- 측정 단위
  - CPU Time, Wall Time, Duration, Errors
