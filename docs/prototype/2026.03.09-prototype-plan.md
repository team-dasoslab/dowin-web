# Dowin 프론트엔드 프로토타입 계획

- [x] **로그인 화면**: ID/PW 입력 및 간단한 인증 시뮬레이션
- [x] **초기 온보딩**: 최초 로그인 시 닉네임 설정 및 비밀번호 변경 권장 토스트

### Phase 2: 핵심 대시보드 (Main Dashboard)

- [x] **워크스페이스 대시보드**:
  - Goal Card: "X에서 Y까지" 목표 가시화
  - 주간 점수판 테이블 (O/X 토글 인터랙션)
  - 나의 지표 요약 및 팀원 현황 뷰 전환
- [x] **Global State**: React Context를 통한 Mock 데이터 통합 관리 (X to Y 데이터 구조 적용)

### Phase 3: 지표 상세 및 관리 (Measure Management)

- [ ] **지표 상세 페이지**: Recharts를 활용한 선행/후행지표 추이 시각화
- [x] **지표 관리**:
  - **가이드형 추가 폼**: 4DX 원칙(예측성/통제력) 팁 및 Dowin 컨텍스트 제공
  - 수정, 보관(Archive), 물리 삭제 기능 구현 완료

### Phase 4: 프로필 및 설정 (Profile & Settings)

- [x] **프로필 관리**: 닉네임 수정, 비밀번호 변경 (마이페이지 통합)
- [x] **계정 관리**: 서비스 탈퇴(계정 삭제) 프로세스 및 컨펌 UI

### Phase 5: 인터랙션 및 폴리싱 (Interaction & Polishing)

- [x] **푸시 알림 시스템**: 매일 밤 9시 알림 기능 구현 및 서비스 워커 연동
- [x] **낙관적 업데이트**: 알림 토글 등 비동기 작업 시 즉각적인 UI 반응 적용
- [x] **UI/UX 폴리싱**:
  - 브라우저 `alert()`를 커스텀 토스트로 전면 대체
  - 리니어 스타일의 애니메이션 및 피드백 강화
- [/] **모바일 반응형**: 세로형 테이블 최적화 및 레이아웃 조정 (진행 중)

## 3. 기술 스택

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS (Notion-like Design)
- **State Management**: React Context API
- **Visualization**: [Recharts](https://recharts.org/)
- **Animation**: `canvas-confetti`, [React Bits](https://reactbits.dev/) (Framer Motion 기반)

### UI 가이드 원칙 (X to Y Guidance)

- 사용자가 목표를 입력할 때 **"X에서 Y까지 [무엇]을 [언제]까지 달성한다"**는 가이드를 힌트나 플레이스홀더로 제공하여 4DX 원칙 준수를 유도합니다.
