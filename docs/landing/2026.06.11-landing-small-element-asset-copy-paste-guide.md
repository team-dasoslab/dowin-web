# Dowin 랜딩 소형 에셋 생성 복붙 가이드

## 1. 이번 기준

이미 선택한 아래 파일들의 질감과 느낌을 기준으로 맞춘다.

```text
public/assets/landing/element-goal-target-orb.jpeg
public/assets/landing/element-mini-scoreboard-card.jpeg
public/assets/landing/element-problem-chat-bubbles.jpeg
public/assets/landing/element-weekly-check-cells.jpeg
```

공통 질감:

- 순백색 배경 위 단일 3D 오브젝트
- matte soft clay/plastic 재질
- 둥근 bevel과 두께감 있는 extruded 형태
- glossy/metal이 아니라 부드러운 반무광
- 파란색, 흰색, 연회색, 검정만 사용
- raised bar, raised dot, raised checkbox처럼 살짝 돌출된 UI 디테일
- 중앙 배치, 넉넉한 여백
- 생성 이미지 자체에는 그림자를 넣지 않음
- 완전 평면 벡터가 아니라 부드러운 3D 제품 렌더
- 각도는 강한 사선이 아니라 얕은 3/4 product render 정도만 허용

생성 결과가 이 4개와 한 세트처럼 보이면 통과다.

## 2. 생성 옵션

- Format: PNG 또는 JPEG
- Background: pure white
- Size: 1024x1024
- Style: matte clay 3D product render
- Text: none
- Camera: centered, shallow front 3/4 view
- Shadow: no shadow

## 3. 매번 공통으로 붙일 스타일 고정 문장

아래 문장은 스타일 기준 설명이다. 실제 작업할 때는 아래 `복붙` 블록만 사용하면 된다.

중요: 각 `복붙` 블록 안에는 이 공통 질감 문장이 이미 중복 포함되어 있다. 공통 문장을 따로 추가로 붙이지 않는다.

```text
Create a small isolated 3D landing-page asset for Dowin. Match this exact visual style: matte soft clay/plastic product render, pure white background, centered object, generous empty margin, rounded bevels, thick extruded shapes, raised UI details, soft ambient occlusion, subtle gray edge shading. Do not include any shadow under or around the object. Use only Dowin primary indigo blue #3A64C7, white, light cool gray, medium zinc gray, and deep black #09090B. The asset should look like the same set as a 3D target icon, stacked checklist cards, chat bubbles, and a small dashboard card. Use a shallow front 3/4 product-render angle, not a steep isometric angle. No readable text.
```

아래 네거티브도 각 `복붙` 블록 안에 이미 중복 포함되어 있다.

## 8. 버릴 기준

아래 중 하나라도 있으면 저장하지 말고 다시 생성한다.

- 선택된 4개 기준 에셋과 질감이 다르다.
- 유리, 금속, 고광택 장난감처럼 보인다.
- 평면 벡터 아이콘처럼 보인다.
- 색이 파랑/흰색/회색/검정에서 크게 벗어난다.
- 보라색, 분홍색, 네온, 다색 그라데이션이 보인다.
- 글자처럼 보이는 가짜 텍스트가 있다.
- 생성 이미지 안에 그림자, drop shadow, contact shadow, cast shadow가 있다.
- 강한 사선/아이소메트릭 시점이다.
- 사람, 손, 캐릭터, 마스코트가 있다.
- 실제 앱 스크린샷처럼 보인다.
