# 브런치 딥다이브 타깃 리서치

작성일: 2026-05-21

## 이 문서의 목적

이 문서는 `브런치만 더 깊게 파보자`는 목적 아래,  
브런치 공개 글에서 Dowin과 직접 맞닿는 신호를 다시 좁혀 본 결과를 정리한 문서다.

이전 라운드에서 이미 확인한 사실이 하나 있었다.

- 넓게 보면 브런치가 제일 유효하다
- 그런데 브런치 안에서도 `운영 일반론`이 너무 많다
- 그래서 검색 키워드를 더 날카롭게 좁히지 않으면 다시 노이즈가 늘어난다

이번 문서는 그 문제를 해결하기 위해 아래를 남긴다.

1. 브런치에서 실제로 유효했던 검색 방식
2. 강한 신호와 약한 신호의 차이
3. 이번 라운드에서 다시 남긴 사람/조직 후보

## 한 줄 논리

`브런치에서 좋은 후보는 "운영이 중요하다"를 말하는 사람이 아니라, "대표 머릿속에 일이 남아 있다", "회의 후 액션이 흐려진다", "미팅 전에 다시 적어둔다" 같은 구체 장면을 말하는 사람이다.`

## 이번 라운드의 검색 방향

이번에는 아래 표현을 중심으로 찾았다.

- `관제탑`
- `대표의 머릿속`
- `회의 전`
- `다시 정리`
- `액션 아이템`
- `다음 회의`
- `누가 무엇을 하기로 했는지`
- `레슨런`
- `데일리스크럼`

반대로 아래처럼 너무 넓은 키워드는 덜 믿었다.

- `체계`
- `운영`
- `협업`
- `소통`
- `생산성`

이 키워드들은 후보를 너무 많이 주지만, Dowin과의 연결은 약했다.

## 먼저 결론

이번 브런치 딥다이브 후 결론은 아래와 같다.

1. `글쓰는 COO`는 생각보다 더 강한 후보였다
2. `셔니`는 여전히 최상위다
3. `정영현 CEO / 코르카`와 `박미영 대표 / 제제미미`는 리더가 기록을 다시 꺼내 개입하는 장면을 보여주는 보조 후보다
4. `알고케어 회의어트 TF`는 여전히 아주 강한 조직 신호다
5. 브런치 안에서도 `운영 일반론`은 계속 버려야 한다

즉 이번 라운드는 `새로운 사람을 많이 건졌다`보다  
`브런치 안에서 진짜 봐야 할 표현이 무엇인지 더 분명해졌다`는 의미가 크다.

## 강한 사람 후보

## 1. 셔니

- 핵심 표현:
  - `CEO의 전략적 의사결정을 돕고 조직 운영을 지원하는 포지션`
  - `회사 흐름을 읽고 실행까지 맡는`
  - `대표가 고민하는 거의 모든 문제를 가장 가까이서 함께 고민하고 해결`
- 왜 강한가:
  - 대표 직속 운영/실행 혼합 역할이 직접 나온다
  - 리더 개입 준비 비용이 사람에게 몰려 있는 구조가 선명하다
- 아직 확인이 필요한 것:
  - 회의 전 상태 재정리 장면
  - 도구 분산 장면
- 현재 판정:
  - `최상위 유지`
- 출처:
  - https://brunch.co.kr/@w-siyeon/1

## 2. 글쓰는 COO

- 핵심 표현:
  - `세 개의 툴을 병행하는 방식`
  - `노션이 훌륭한 도서관이라면, 우리에겐 매일의 전투를 치러낼 관제탑이 필요했다`
  - `이 세 가지만으로도 대표의 머릿속에서 일이 빠져나가기 시작했다`
- 왜 강한가:
  - 이건 단순한 체계론이 아니라, `기록 도구는 있는데 운영 관제탑이 없다`는 말이다
  - Dowin이 노리는 `기록은 있지만 리더 운영 비용이 크다`는 장면과 거의 정확히 맞닿는다
- 아직 확인이 필요한 것:
  - 주간 회의/점검 리듬
  - 실제로 대표 또는 운영 리더가 어떤 식으로 재정리했는지
- 현재 판정:
  - `재상승`
- 출처:
  - https://brunch.co.kr/@coo-notes/23

## 브런치에서 강한 신호와 약한 신호

## 강한 신호

아래 표현이 직접 나오면 강하다.

- `대표의 머릿속에서 일이 빠져나가기 시작`
- `관제탑이 필요`
- `미팅 전에 ... 미리 적어둔다`
- `회의 액션 아이템 관리가 잘 되지 않는다`
- `회의 후 내용 정리가 잘 되지 않는다`
- `누가 무엇을 하기로 했는지 서로 이해가 다르다`

## 약한 신호

아래 표현만 있으면 약하다.

- `체계가 필요하다`
- `운영이 중요하다`
- `소통이 안 된다`
- `회의가 많다`

이건 문제는 말하지만 `왜 Dowin인가`까지는 잘 안 이어진다.

## 브런치 전용 검색 규칙

다음에 브런치를 더 깊게 볼 때는 아래 순서가 맞다.

1. `대표의 머릿속`, `관제탑`, `액션 아이템`, `회의 후`, `미리 적어둔다` 같은 구체 표현부터 찾는다
2. 글 안에 `노션`, `슬랙`, `데일리스크럼`, `회의록`, `기록`, `담당자`가 같이 나오는지 본다
3. `리더가 직접 다시 본다`, `다시 묻는다`, `미리 적어둔다` 같은 개입 장면이 있는지 본다
4. 일반론이면 버리고, 직접 장면이 있으면 남긴다

## 현재 결론

브런치 딥다이브 결과, 지금 가장 강한 두 축은 여전히 아래다.

1. `셔니`
2. `글쓰는 COO`

보조 축은 아래다.

3. `정영현 CEO / 코르카`
4. `박미영 대표 / 제제미미`
5. `알고케어 회의어트 TF`

즉 브런치만 더 깊게 파도 핵심은 크게 바뀌지 않았다.

대신 `글쓰는 COO`의 신호가 예상보다 강해졌고,  
앞으로는 `대표 머릿속`, `관제탑`, `미리 적어둔다`, `액션 아이템 관리` 같은 표현을 최우선으로 보는 편이 맞다는 점이 분명해졌다.
