# 한국인 인터뷰 후보 찾기 가이드

작성일: 2026-05-20

## 이 문서의 목적

이 문서는 Dowin의 현재 1순위 ICP를 검증하기 위해,  
한국인 인터뷰 후보를 어디서 찾고 어떤 키워드로 검색할지 실행 기준을 정리한 문서다.

목적은 아래 세 가지다.

1. 인터뷰 후보를 찾을 채널 우선순위를 고정한다
2. 검색 키워드를 문제 중심으로 정리한다
3. 실제로 연락할 후보를 추리는 기준을 명확히 한다

이 문서는 `실행용 후보 탐색 가이드`다.

## 현재 기준 ICP

지금 기준으로 가장 먼저 검증할 ICP는 아래에 가깝다.

- 3~15인 소규모 팀
- 대표 직속 / 운영 / 기획 혼합 역할이 있음
- 주간 회의나 실행 점검이 있음
- Notion, 카카오톡, 문서, 시트, Slack 같은 조합으로 운영 중임
- 회의 전 상태 정리, follow-up 추적, 리더 직접 chase 문제가 있음

## 어디서 찾을지

### 1. 브런치

가장 먼저 볼 채널이다.

이유:

- 한국어로 운영 pain를 길게 쓰는 사람이 많다
- 대표, 운영 리더, 기획/운영 혼합 역할이 자기 문제를 직접 설명하는 글이 많다
- 글을 읽고 연락할 수 있어 인터뷰 요청 명분이 자연스럽다

주요 대상:

- 스타트업 대표
- COO 또는 운영 리드
- 대표실 / 운영기획 역할
- Chief of Staff 성격의 역할

### 2. 링크드인 게시글 검색

프로필 네트워킹보다 `게시글 검색` 위주로 본다.

이유:

- 한국 스타트업 실무자가 운영, 실행, follow-up 문제를 공개적으로 많이 쓴다
- 영어 직함과 한국어 pain 표현이 함께 섞여 있어 후보 찾기가 쉽다

주요 대상:

- 창업자
- COO
- BizOps
- Operations Manager
- Chief of Staff
- Founder Office / 대표실 역할

### 3. X

짧지만 감정이 드러나는 pain를 보기 좋다.

이유:

- 대표나 운영 리더가 `결국 내가 다시 챙긴다` 같은 문제를 짧게 많이 올린다
- 완성된 글보다 생생한 불만 표현을 찾기 좋다

주요 대상:

- 스타트업 대표
- 초기팀 운영 담당자
- 사업/운영/기획 혼합 역할

### 4. 스타트업 커뮤니티 / 오픈채팅 / 슬랙

이유:

- 인터뷰 모집글을 올리기 좋다
- 1:1 콜드 DM보다 부담이 적다

주요 대상:

- 스타트업 운영자 모임
- 창업자 모임
- 실무자 커뮤니티
- ops / biz 중심 커뮤니티

### 5. 채용공고 역추적

이유:

- 어떤 팀이 지금 운영 pain를 갖고 있는지 간접적으로 볼 수 있다
- `대표실`, `BizOps`, `Chief of Staff`, `운영기획`을 채용 중인 팀은 ICP와 맞을 가능성이 높다

활용 방식:

- 채용공고로 팀을 찾는다
- 회사 규모와 역할 설명을 읽는다
- 그 팀의 대표나 운영 담당자를 찾아 인터뷰 요청한다

## 채널 우선순위

아래 순서로 보는 편이 맞다.

1. 브런치
2. 링크드인 게시글 검색
3. X
4. 커뮤니티 / 오픈채팅
5. 채용공고 역추적

이유:

- `문제를 이미 글로 말한 사람`에게 가는 것이 가장 자연스럽다
- 직함만 보고 무작위로 보내는 방식보다 반응률이 높다

## 채널별 검색 방법

### 브런치

검색 예시:

```text
스타트업 운영
대표실
운영기획
COO
팀 운영
회의
팔로업
실행
정렬
우선순위
```

볼 포인트:

- `회의는 하는데 실행이 흐려진다`
- `결국 대표가 다시 챙긴다`
- `문서/메신저/회의가 따로 논다`

### 링크드인

검색 예시:

```text
"Chief of Staff" 운영
"COO" 스타트업
"Operations Manager" 스타트업
"BizOps" 스타트업
대표실 운영
운영기획 스타트업
```

게시글에서 볼 표현:

- alignment
- follow-up
- weekly sync
- operating rhythm
- shared context
- visibility
- execution

### X

검색 예시:

```text
스타트업 운영
대표가 다 챙겨
회의만 하고
우선순위 정리
실행 관리
팔로업
노션 운영
카톡 운영
```

볼 포인트:

- 짧아도 감정이 드러나는 글
- `너무 정신없다`, `내가 또 챙겼다`, `회의했는데 남는 게 없다` 같은 표현

### 채용공고

검색 키워드:

```text
운영기획
사업운영
BizOps
Chief of Staff
대표실
운영 매니저
프로젝트 운영
```

공고에서 읽을 것:

- `우선순위 정리`
- `cross-functional coordination`
- `대표/경영진 아젠다 관리`
- `실행 tracking`
- `follow-up`
- `운영 리듬 관리`

이런 문구가 있으면 그 팀은 pain가 있을 가능성이 높다.

## 좋은 후보 판별 기준

아래 조건 중 4개 이상 맞으면 인터뷰 후보로 본다.

- 팀이 3~15명 정도다
- 창업자나 리더가 아직 운영을 직접 챙긴다
- 운영/기획/대표실 역할이 있다
- 회의, 실행, 후속조치가 반복된다
- Notion, 카카오톡, 문서, 시트, Slack 같은 도구 조합이 보인다
- `정리`, `공유`, `추적`, `팔로업`, `우선순위` 같은 말을 자주 쓴다
- 너무 엔터프라이즈스럽지 않다
- 너무 개발 PM 툴 중심 팀 같지 않다

## 후보 정리 항목

후보를 찾으면 아래 항목으로 바로 정리한다.

```md
- 이름
- 회사/팀
- 역할
- 어디서 찾았는지
- 문제 단서 문장
- 왜 ICP에 맞는지
- 우선순위 (1/2/3)
- 연락 가능 채널
- 연락 여부
- 답장 여부
- 인터뷰 일정
```

## 첫 주 목표

처음부터 크게 벌리지 말고 아래 정도면 충분하다.

```md
- 브런치 글 10개 읽기
- 링크드인/X 글 10개 보기
- 후보 15명 리스트업
- 그중 8~10명에게 인터뷰 요청 보내기
- 3명 대화 성사
```

## 가장 추천하는 시작 순서

1. 브런치에서 `스타트업 운영`, `운영기획`, `대표실`, `팀 회의`, `우선순위` 검색
2. 링크드인에서 `Chief of Staff`, `COO`, `BizOps`, `운영기획 스타트업` 게시글 검색
3. `문제 글을 쓴 사람`만 후보로 저장
4. 후보 10명이 모이면 인터뷰 요청 보내기

## 현재 기준 한 줄 결론

`지금은 넓은 한국 시장을 찾는 것이 아니라, 회의 전 상태 정리와 리더 직접 chase 문제가 이미 드러난 한국인 운영 리더 후보를 브런치, 링크드인 게시글, X에서 먼저 찾는 편이 맞다.`
