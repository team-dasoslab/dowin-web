# Dowin ICP Notes

작성일: 2026-05-13

## 이 폴더의 목적

이 폴더는 경로 이름만 `consulting`으로 남아 있는 레거시 폴더다.  
현재 기준으로는 `Dowin ICP 탐색`에 직접 필요한 메모만 남겨 둔 상태다.

핵심 목적은 아래 세 가지다.

1. 어떤 고객 구조가 Dowin와 맞는지 좁힌다
2. 세그먼트별 문제 문장을 다듬는다
3. 기존 리서치를 `확정 결론`이 아니라 `가설 자산`으로 관리한다

즉 지금 이 폴더는 `컨설팅 사업 문서함`이 아니라  
`Dowin ICP 탐색용 보조 노트 폴더`로 읽는 편이 맞다.

## 현재 운영 원칙

- 이 폴더의 문서는 `확정된 ICP 결론`이 아니다
- 모두 `직접 인터뷰 전/중에 쓰는 가설 자산`으로 다룬다
- 다만 우선순위는 있다
- 1순위: founder-led 또는 대표 직속 운영/기획 혼합 팀
- 2순위: 반복 프로젝트 운영 팀
- 3순위: 다점포/다지점 본사 운영 팀
- 새로운 문서를 추가할 때도 `Dowin ICP 탐색`에 직접 필요한 경우만 남긴다
- 폴더 이름 `consulting`은 과거 히스토리 때문에 유지할 뿐, 현재 목적을 설명하는 이름은 아니다
